## Super Fast Setup

### 1. Install Node.js
If you don't have it: Download from [nodejs.org](https://nodejs.org)

### 2. Open Terminal
- **Windows**: Press `Win + R`, type `cmd`, Enter
- **Mac**: Press `Cmd + Space`, type "terminal", Enter  
- **Linux**: Press `Ctrl + Alt + T`

### 3. Navigate to Project
```bash
cd path/to/valentine-pixel
```

### 4. Install Dependencies
```bash
npm install
```
(Takes 1-2 minutes)

### 5. Start the Site
```bash
npm run dev
```

### 6. Open in Browser
Visit: `http://localhost:5173`
